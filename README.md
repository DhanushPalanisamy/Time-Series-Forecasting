## Overview
This project forecasts future sales using ARIMA and displays results on a web interface.

## Tech Stack
- Python
- Flask
- Pandas
- ARIMA (statsmodels)
- Plotly

## How to Run

1. Install dependencies  
   pip install -r requirements.txt

2. Train model  
   python train_model.py

3. Run app  
   python app.py

4. Open browser  
   https://time-series-forecasting-5710.onrender.com/

## Output
Interactive graph showing historical sales and future forecast.
